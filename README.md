# 全屋设计软件

一个面向普通业主的网页端 2D/3D 全屋设计工具。当前版本使用 React、TypeScript、Vite、react-konva、Konva 和 Three.js 构建，采用本地浏览器优先的方式保存方案，并为后续云保存、手机采集、AI 设计、高级渲染和商品模型库预留数据结构。

## 主要功能

- 户型模板：内置一居室、两居室、三居室起步方案。
- 户型图上传：支持 PNG、JPG、JPEG、WebP，可识别到独立图层或仅作为底图参考。
- 识别图层：上传后可生成候选墙体图层，支持选择、Ctrl 多选、删除、恢复、合并、补墙和写入正式方案。
- 墙体绘制：默认一段一段画，可切换连续绘制，支持 Esc 取消起点、Enter 结束连续绘制。
- 门窗与家具：支持在墙上放置门窗，家具库包含客厅、卧室、餐厅、厨房、卫浴、书房、收纳、儿童房、阳台、玄关和灯饰等分类。
- 家具材料系统：支持家具子类、风格标签、材质库、收藏、推荐、组合家具、材质刷和商品字段。
- 组合家具：内置客厅、卧室、餐厅、儿童房、玄关等套装，拖入后可整体移动、旋转、复制、删除和取消组合。
- 材质刷：可把木饰面、布艺、皮革、金属、玻璃、石材、陶瓷、塑料等材质应用到家具或房间区域。
- 商品字段：家具可填写品牌、系列、SKU、参考价和商品链接，参考价可进入预算。
- 房屋面积：可手动填写房屋总面积，也可添加房间标签作为参考。
- 房间区域：可绘制多边形房间区域，自动计算面积，并为房间选择地面、墙面、顶面材料。
- 材料预算：根据房间面积、墙高、材料单价、损耗率和家具参考价生成本地预算估算。
- 2D/3D 视图：平面编辑与 Three.js 3D 预览可切换，支持灯光、材质模式、环境背景、房间地面材质、相机预设和阴影。
- 本地 3D 效果图：支持俯斜总览、斜角效果、正面展示、顶视平面、漫游视角，导出倍率支持 1x、2x、3x。
- 本地保存：方案保存到浏览器 IndexedDB，刷新后可恢复。
- 导出能力：支持导出 2D PNG、3D PNG、方案 JSON、预算 CSV 和 HTML 交付报告。

## 推荐流程

1. 新建空白方案，或从一居室、两居室、三居室模板开始。
2. 上传真实户型图，优先选择“识别到图层”，在右侧识别图层面板中删除误识别墙、补墙和合并短墙。
3. 将确认后的识别墙写入正式方案，或者使用墙体工具手动画墙。
4. 使用“房间”工具沿房间边界点击，最后点击首点闭合区域，生成自动面积。
5. 在右侧属性栏为房间区域填写名称、手动面积覆盖和材料。
6. 从家具库拖入单个家具或组合家具，按房间、风格、材质、收藏和推荐筛选。
7. 使用“材质”工具选择材质刷，点击家具或房间区域快速换材质。
8. 为家具填写品牌、系列、SKU、参考价和链接，按需进入预算。
9. 切换 3D 预览检查空间效果，并在“3D 效果图”面板中选择环境、灯光、相机和导出倍率。
10. 保存方案，并按需要导出 PNG、JSON、CSV 或 HTML 报告。

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开：

```text
http://127.0.0.1:5173/
```

构建生产版本：

```bash
npm run build
```

## 长期路线图

- V5：房间区域、自动面积、材料预算、CSV 清单、HTML 交付报告。
- V6：本地轻量 3D 真实感升级，包括环境背景、灯光预设、相机机位、房间材料和高清截图。
- V7：户型识别增强，继续提升墙体、房间轮廓、门窗候选和批量修正能力。
- V8：家具材料系统，扩展模型库、收藏、组合家具、材质刷和商品字段。
- V9：户型识别补全，增强外框、阳台、门窗候选、房间轮廓和质量报告。
- V10：导入导出增强，支持 PDF 报告、SVG、DXF 草案、GLB/OBJ 草案和打印布局。
- V11：手机与云端预留，支持手机采集数据导入、云同步接口草案、分享链接草案和多人协作草案。
- V12+：AI 自动布局、AI 风格方案、AI 渲染任务、施工报价和真实商品接入。

## 当前范围

当前版本仍是本地优先的网页应用，不接真实账号、云保存、支付、真实商品库、真实 AI 云渲染或手机 LiDAR/AR 扫描服务。家具商品字段只作为本地方案数据保存，不联网校验商品链接。3D 效果图是本地实时预览与截图，不等同于照片级云渲染。

# 全屋设计软件

一个面向普通业主的网页端 2D/3D 全屋设计工具。当前版本使用 React、TypeScript、Vite、react-konva 和 Three.js 构建，支持户型绘制、户型图底图上传、自动生成墙体、家具布置、本地保存、平面图导出和简易 3D 预览。

## 主要功能

- 户型模板：内置一居室、两居室、三居室起步方案。
- 手动画墙：支持网格吸附、墙体选择、端点拖拽和墙长输入。
- 户型图上传：支持 PNG、JPG、JPEG、WebP 作为半透明底图。
- 比例标定：通过两点标定真实长度，校准画布比例。
- 自动生成墙体：基于上传底图识别深色墙线，生成可编辑墙体。
- 门窗元素：支持在墙体上放置门、窗，并沿墙体方向对齐。
- 家具库：包含客厅、卧室、餐厅、厨房、卫浴、书房、收纳等分类家具。
- 快捷编辑：选中家具后可复制、粘贴、旋转、删除，并支持方向键按网格移动。
- 方案信息：可填写房屋总面积，并查看房间标注面积合计。
- 本地保存：方案保存到浏览器 IndexedDB，刷新后可恢复。
- 导出能力：支持导出 PNG 平面图、方案 JSON 和当前视角 3D 效果图。
- 3D 预览：将 2D 墙体、门窗和家具生成简易 Three.js 预览场景。

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开：

```text
http://127.0.0.1:5173/
```

构建生产版本：

```bash
npm run build
```

## 使用建议

如果要根据真实户型图开始设计，建议先新建空白方案，再上传户型图。上传后可以先进行比例标定，再点击“自动生成墙体”。自动识别会优先提取黑色或深灰色墙线，复杂户型图中的文字、家具、水印和尺寸线可能造成误判，生成后仍建议手动微调。

## 当前范围

当前版本优先保证网页端编辑体验和本地数据结构可扩展。暂不包含云端账号、真实商品库、报价系统、照片级渲染和桌面端封装。后续可以继续扩展云保存、手机端查看、真实 3D 模型导出、效果图渲染和装修预算等能力。
